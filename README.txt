Для сохранения изменений:
Открыть проект в VS Code
→ Изменить 
1. Для замены картинки, нужно посмотреть на сайте имя картинки.
Для новой картинки установить соответствующее имя.
После изменить формат картинки на .webp.
Для изменения формата нужно открыть сайт: squoosh.app.
Перетащить туда картинку, выбрать опцию WebP и сохранить.
Новую картинку добавить в проект вместо старой.
2. Для редактирования текста нужно открывать файлы: cases.html или index.html (главная страница).
После внесения изменений нажать Ctrl + S (для сохранения изменений).
→ Commit 
По завершении внесения изменений нужно нажать на иконку под лупой в левом боковом меню.
Написать комментарий к изменениям в поле для ввода
→ Push
Далее нужно нажать на стрелочку на кнопке Commit
Выбрать опцию Commit & Push
→ Готово!
Через несколько минут изменения будут отражены на сайте
По ссылке: https://feinp51.github.io/Bakhroma_Homestaging/
